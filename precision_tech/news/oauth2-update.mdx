### What is changing?

Your application must update to the **OAuth 2** protocol by **May 1, 2022** to stay connected to the JDLink Machine Data API and support your customers' connections to their John Deere account.

- Technical documentation on the OAuth 2 update can be found at [developer.deere.com](https://developer.deere.com).
- The API endpoints used with OAuth 2 user tokens can be found [here](#).
