  This integration ensures that dealer websites consistently display the most current product and marketing details available on Deere.com . By utilizing these APIs, dealers can keep their digital presence accurate and up-to-date, improving both operational efficiency and customer experience.
  Currently, the APIs expose endpoints for Product Categories and detailed Product Information, including specifications, features, and benefits. Additional marketing content—such as digital assets, brochures, offers, and delta-based data retrieval—will be introduced incrementally to support evolving business needs.

# Endpoints

## Retrieve all product categories
Fetch all product categories with optional hierarchical children (up to 3 levels deep).

GET
/caas/v1/ddf/categories

<AccordionGroup>
  <Accordion title="Parameters">
  | Parameter       | Type    | Description                                                                | In    |
  | --------------- | ------- | -------------------------------------------------------------------------- | ----- |
  | locale          | string  | Language and region code for localized content (e.g., en_US, de_DE, fr_FR) | query |
  | includeChildren | boolean | Include child categories in the response hierarchy (up to 3 levels deep)   | query |
  </Accordion>
  <Accordion title="Response Details">
  | Field      | Type    | Description     |
  | ---------- | ------- | --------------- |
  | taxonomies | object  | Categories data |
  </Accordion>
