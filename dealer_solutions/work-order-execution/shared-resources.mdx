## Overview

This page groups the supporting endpoints that are used across Work Order Execution workflows.

<CardGroup cols={2}>
  <Card title="Customers and features" icon="users">
    Retrieve DBS customers and feature toggles for the authenticated context.
  </Card>
  <Card title="Equipment and inspections" icon="tractor">
    Look up inspections, machine smart info, warranty details, and general segment metadata.
  </Card>
  <Card title="Images and files" icon="image">
    Upload, replace, download, and delete shared image or file attachments.
  </Card>
  <Card title="Translations and diagnostics" icon="language">
    Access locale data, update translations, check undercarriage data, and verify service health.
  </Card>
</CardGroup>

<Note>
  Authentication and base URL details are documented on the main `Work Order Execution API` overview page.
</Note>
